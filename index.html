OpenAI API 转发

本站是对 OpenAI API 的转发，以解决国内访问 OpenAI API 的问题。网站源码请见 GitHub。

使用方法

需要使用的第三方客户端支持配置 API HOST，按照客户端的说明，把 https://openai.saplf.top 填入即可。

ChatGPT 第三方客户端

  • AMA / 安卓端 ChatGPT 应用
  • OpenCat / iOS&Mac ChatGPT 应用
  • ChatBox / 桌面端 ChatGPT 应用
  • ChatGPT Web / 使用 Next 编写的 ChatGPT 网页版
  • ChatGPT Mirror / ChatGPT 网页版镜像
